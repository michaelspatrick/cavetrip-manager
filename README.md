# CaveTrip Manager

Open-source cave trip planning, waiver, landowner, and safety management software for NSS grottos and cave exploration organizations.

## Mission

CaveTrip Manager helps grottos manage cave trips, digital waivers, landowner relationships, participant safety, callouts, rescue packets, and trip reports in one self-hosted web application.

## Project Status

Early design and foundation phase.

## Core Concepts

- Multi-grotto support
- Trip-centered workflow
- Role-based permissions
- Digital waiver templates and signatures
- Landowner and cave records
- Emergency contacts and medical notes
- Callout tracking
- Emergency rescue packet
- Email and Pushover notifications
- Trip reports and audit logging

## Technology Goals

- PHP 8.1+
- MySQL / Percona MySQL / MariaDB compatible
- Bootstrap 5
- Composer-based dependencies
- Self-hostable by volunteer organizations
