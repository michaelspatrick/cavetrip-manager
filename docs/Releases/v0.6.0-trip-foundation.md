# v0.6.0 Trip Foundation

Adds the first trip-management module.

## Added

- Trip list page at `/trips`
- Create trip page at `/trips/create`
- Manage trip page at `/trips/show?id=...`
- Edit trip page at `/trips/edit?id=...`
- Trip cancellation workflow scaffold
- Capacity fields: minimum attendees, maximum attendees, waitlist enabled
- Visibility fields: core group, selected members, invite link, private
- Callout time field
- Trip workflow checklist scaffold
- Share token generation for future signup links

## Notes

Participant signup, signature collection, waiver finalization, and email/Pushover notifications are still pending future releases.
