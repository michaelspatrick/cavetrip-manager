# v0.8.0 - Waiver Templates

This release adds the first waiver administration module.

## Added

- Waiver template list, create, and edit screens.
- Admin navigation link for Waiver Templates.
- Waiver template service with grotto-scoped CRUD operations.
- Placeholder helper panel for common merge fields.
- Starter HTML waiver template text.
- CSS improvements for the HTML editor and placeholder chips.
- Stable `/health` route remains available for deployment checks.

## After Uploading

Run:

```bash
php migrate.php
```

Then visit:

```text
/waiver-templates
/waiver-templates/create
/health
```
