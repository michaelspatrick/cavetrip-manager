# v0.3.0 Auth Foundation

This release adds the first usable account layer for CaveTrip Manager.

## Added

- GET/POST routing support.
- Session handling with secure cookie options.
- CSRF token generation and validation.
- Login/logout workflow.
- AuthService and UserService.
- Dashboard requiring authentication.
- User list and create-user screen for admins.
- CLI admin creation tool.
- Password reset and login event support tables.

## Create the first account

Run migrations first:

```bash
php migrate.php
```

Create a super admin:

```bash
php tools/create_admin.php \
  --name="Michael Patrick" \
  --email="you@example.com" \
  --password="CHANGE_ME_STRONG_PASSWORD"
```

Or create a grotto admin and grotto in one command:

```bash
php tools/create_admin.php \
  --name="Michael Patrick" \
  --email="you@example.com" \
  --password="CHANGE_ME_STRONG_PASSWORD" \
  --grotto-name="Six Ridges Grotto" \
  --grotto-slug="six-ridges"
```

Then visit `/login`.

## Notes

This is still scaffolding. Password reset email delivery, user editing, role conversion, and grotto settings are planned for upcoming releases.
