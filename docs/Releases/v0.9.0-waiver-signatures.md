# v0.9.0 - Waiver Signatures and Final Waiver View

This release adds the first end-to-end waiver workflow foundation.

## Added

- Participant signature tokens.
- Mobile signature capture page at `/sign?token=...`.
- Signature saving to `trip_participants.signature_data`.
- Trip dashboard signature links for participants.
- Waiver finalization action from the trip dashboard.
- Generated waiver archive records.
- Public permanent final waiver view at `/waivers/view?token=...`.
- Print-friendly final waiver page.

## Notes

- Email delivery is intentionally deferred to the notification release.
- Finalization currently requires all active participants to have signed.
- Signature data is stored as a PNG data URL in the database for now.
