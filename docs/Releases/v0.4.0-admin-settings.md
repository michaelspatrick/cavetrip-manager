# v0.4.0 Admin Settings

This release adds the first practical admin UI layer on top of the auth foundation.

## Added

- Admin dashboard stats
- Grotto settings page at `/admin/grotto/settings`
- Grotto contact fields editing
- Logo URL support
- Logo upload support to `public/uploads/logos`
- Admin navigation links
- `GrottoService`
- `DashboardService`
- Audit logging for grotto settings updates

## Admin access

1. Run migrations.
2. Create the first admin using `tools/create_admin.php`.
3. Visit `/login`.
4. After login, visit `/dashboard` or `/admin/grotto/settings`.

## Notes

This release intentionally does not add trip creation yet. The next milestone should add the landowner and cave management screens, followed by trip creation.
