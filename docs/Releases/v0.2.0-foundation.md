# v0.2.0 Foundation Release

This release turns the initial project scaffold into a runnable PHP foundation.

## Added

- Basic application bootstrap
- `.env` loading
- PDO database connection
- Minimal router and view renderer
- Public landing page
- `/health` JSON endpoint
- CLI migration system
- Initial database migrations
- Service stubs for tokens, trip numbers, migrations, and audit logging
- Starter CSS
- Updated README with local setup instructions

## Purpose

This release intentionally avoids building trip creation screens yet. The goal is to freeze the initial naming conventions and create a reliable base before adding user authentication, permissions, and trip workflows.

## Suggested commit

```bash
git add .
git commit -m "Add foundation application skeleton and migrations"
git tag v0.2.0-foundation
```
