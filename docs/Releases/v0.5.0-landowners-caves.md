# v0.5.0 Landowners and Caves

This release adds the first operational data management module for CaveTrip Manager.

## Added

- Landowner list, create, and edit screens.
- Cave list, create, and edit screens.
- Cave records can be linked to landowners.
- Cave records include restricted access fields for GPS, access directions, parking notes, gate/lock codes, and sensitive notes.
- Audit log entries for creating and updating landowners and caves.
- Navigation links for grotto admins.
- Migration `006_add_cave_access_fields.php`.

## Security Notes

- Cave management pages require `super_admin` or `grotto_admin`.
- Landowner pages are available to managers for now, but sensitive landowner/cave exposure will be tightened further when trip workflows are added.
- Guest users do not have access to cave or landowner management.

## Upgrade

Run:

```bash
php migrate.php
```

Then visit:

```text
/landowners
/caves
```
