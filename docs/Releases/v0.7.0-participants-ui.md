# v0.7.0 Participants + UI Polish

Adds the first participant signup layer and a major CSS cleanup.

## Added

- Trip participant service
- Trip leader participant roster on trip dashboard
- Add walk-in / late participant form
- Remove participant from active roster
- Public tokenized trip signup page
- Required emergency contact fields
- Optional medical/rescue-packet fields
- Better cards, tables, forms, badges, buttons, progress bar, and mobile layout

## Routes

- `POST /trips/participants/add?trip_id=ID`
- `POST /trips/participants/remove?trip_id=ID`
- `GET /trip/signup?token=TOKEN`
- `POST /trip/signup?token=TOKEN`

## Notes

Signature capture and email notifications are intentionally not included yet. This release prepares the roster and safety data foundation needed for waivers, callouts, and rescue packets.
